SIMPLE HTML PAGE!

By default this page is not public. What we need is anyone trying to access the apis should atleast be able to see this page without any need of authentication.

http .authorizeRequests() .antMatchers("/", "index", "/css/*", "/js/*") .permitAll() .anyRequest() .authenticated() .and() .httpBasic();

Using this we can make certain resources as public, easily accessible without any authentication.
